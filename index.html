Welcome!

This is the home of H#: An open source, functional, math-oriented scripting language. It was made with readability and friendliness to math enthusiasts in mind, and aims to be strongest when trying to make short programs with self-explanatory code.

This page is to share the strengths, weaknesses, documentation, and tutorials on H#.




Concise, Weakly-Typed, and Purely Functional

let primes = filterprime [2..100]     where filterprime[] = [], filterprime X =         (head[X]) : filterprime (filter[L => L % (head[X]) /= 0, tail[X]]);

H# is purely functional, meaning it enforces immutable values, and no side-effect-producing functions. Referential transparency was essential when creating H#, but so was friendliness to the user. Operations like getstr and getnum will return a valid string, unlike Haskell. However, any IO operation requires a world parameter, so the code stays referentially transparent.
